This page contains links to PDF versions of additional documentation for the book Compiling with C# and Java by P.D. Terry (Pearson Education, 2005)

*** This version of the Resource Kit is designed for case studies that would match the "instantiable" release of Coco/R. In that release, the generated classes do not export static methods only, as is the case for nearly all the case studies discussed in the text book. In many cases the differences are fairly minor, and the code has been written to minimize the differences.

You will need Adobe Acrobat to read these files.

  • The preface (61kB)
  • The table of contents and chapter objectives (82kB).
  • A description of the simple language Parva (level 1) (99kB).
  • A description of the simple language Parva (level 2) (118kB).
  • A description of the simple language C# Minor (158kB).
  • A user manual for the extended version of Coco/R (264kB).
  • A summary of the differences between the Linz version of the compiler generator Coco/R and the extended version of Coco/R used for the case studies in the book (76kB).
  • The text of the GNU General Public Licence that applies to the distribution of various of the software in the Resource Kit (63kB).
  • The original report on Coco/R by Mössenböck (1990) (72kB)
  • Data structures in Coco/R by Mössenböck (1990) (170kB)
  • LL(1) conflict resolution in a Recursive Descent Compiler Generator by Albrecht Wöß, Markus Löberbauer, Hanspeter Mössenböck. (110kB)
